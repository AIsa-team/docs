# Supported AI Models

AIsa’s Unified LLM Gateway provides access to 50+ state-of-the-art AI models through a single API endpoint. Instead of managing multiple API keys, billing accounts, and rate limits across different providers, you can access the entire ecosystem with one integration.

Our gateway supports models across all major modalities, including text generation, vision understanding, image generation, and audio processing.

## Routing & Fallbacks

AIsa automatically handles rate limits and infrastructure failovers behind the scenes. When you request a model, our gateway routes your request to the optimal upstream provider or cloud deployment to ensure high availability and low latency. If a specific model deployment experiences downtime, AIsa can automatically fallback to alternative deployments of the same model without interrupting your application.
